# Loop Tasks Program

This repository contains a beginner-friendly Python program that demonstrates the use of loops in Python. It covers **for loops**, **while loops**, **break & continue**, **string iteration**, **multiplication table**, **range with steps**, **conditional logic inside loops**, and **real-world loop examples**.
